# saku-vim

## command

```
make build : コンテナをビルド
make run : コンテナを実行
```

## vim command

```
:PlugInstall  : plugin.vimにあるプラグインをインストール
:PlugUninstall: plugin.vimから削除されたプラグインorコメントアウトされたプラグインをアンインストール
```

| map       | result                                  |
| --------- | --------------------------------------- |
| gd        | 定義ジャンプ                            |
| <Space>p  | ファイル検索（のはず）                  |
| <Space>fi | ファイルツリーを開く                    |
| <Space>tb | 下にターミナルを開く                    |
| <Space>tr | 右にターミナルを開く                    |
| <c-e>     | window サイズの調整（後で移動に変える） |
